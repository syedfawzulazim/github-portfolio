# Github Portfolio
This is a Github Portfolio site, which fetches all the necessary data from my github account using GraphQL and NextJs is used for client-site.

## Ingredients: 
  - React
  - NextJs
  - GraphQL
  - Tailwindcss
  - TypeScript
